# Fonts Sync

Read the Figma file linked in `specs/ui/fonts/_map.md` and sync font definitions.

- Extract font definitions from Figma.
- Update `specs/ui/fonts/_map.md` to reflect the current fonts.
- For any custom font file in `specs/ui/fonts/`, update it if the corresponding Figma node has changed.
- Report all changes made and stop for confirmation.
