# Icons Sync

Read the Figma page or section linked in `specs/ui/icons/_map.md` and sync the icon inventory.

- Extract the icon inventory from Figma.
- Update `specs/ui/icons/_map.md` to reflect the current icon set.
- For any custom icon file in `specs/ui/icons/`, update it if the corresponding Figma node has changed.
- Report all changes made and stop for confirmation.
